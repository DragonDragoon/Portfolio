Part 3 - Help Document

This help document assumes the Windows operating system is being used. It offers the simplest way to install IUPLua, which will be described below, and we have already prepackaged the necessary files needed to run the program, libraries and all. This program was written in Lua 5.1.

To run the program in Windows:

  1. Unpack the folder (Project_Part_3-Source) from the compressed file (Part_3_Source.rar) with and IUPLua and LPeg libraries to a directory.
  2. Run the LuaConsole: Project_Part_3-Source/LuaConsole/iuplua51.exe
  3. Load the calculator code: In Lua Console, Load... -> Project_Part_3/main.lua -> Open
  4. Execute code: Click “Execute (F5)”.
  5. The program should run and look like the following:

  1. Click OK and the calculator GUI should be showing like the following:
